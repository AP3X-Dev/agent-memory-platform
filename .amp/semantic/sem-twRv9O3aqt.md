Vertically integrated slices are the load-bearing implementation pattern: each slice (capability) is fully wired end-to-end through every layer (shared types → integration client → tool → graph node → router → SSE event → UI widget mount → tests → parity check) before the next slice begins. No layered horizontal builds.
